## 快速起步

安装：

```bash
npm i
# OR
cnpm i
```

运行：

```bash
npm run dev
# OR
npm run serve
```

打包：

```bash
npm run build
```

自动发布：

```bash
bash publickey.sh

sudo npm run push "提交的说明文案，注意千万不能有空格"
```
