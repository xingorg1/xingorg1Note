¶ 求列表中的平均数

¶ 方法一：通过len做除法

scores1 =  [91, 95, 97, 99, 92, 93, 96, 98]
sum = 0

for score in scores1:
    sum = sum + score
    average = sum/len(scores1) # 通过len方法获取数组长度
print('平均成绩是：{}'.format(average))
# 平均成绩是：95.125
