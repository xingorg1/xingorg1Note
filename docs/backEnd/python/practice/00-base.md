## 九九乘法表
print和for循环知识点

`print( x,end = '  ' )` 不换行

`print('  ')` 换行
```py
for i in range(1,10):
    for j in range(1,i+1):
        print( '%d X %d = %d' % (j,i,i*j),end = '  ' )
    print('  ')
```
